# 清热药
>《黄帝内经》记载“热者寒之”，《神农本草经》也说“疗热以寒之”，足以说明清热药的药性寒凉，它们能沉降入里，通过清热泻火，凉血，解毒及清虚热等不同作用，使得里热得到清解。  
又因为这类药物多寒凉，易伤脾胃，所以脾胃气虚，食少便溏者慎用。不过，现代药理研究表明，部分清热药物有抗肿瘤，抗变态反应及镇静，降血压等作用
