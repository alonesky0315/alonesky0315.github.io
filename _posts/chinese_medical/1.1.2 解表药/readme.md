# 解表药

>表证常见的症状：恶寒，头痛发热，鼻塞咳嗽等  
解表药分为发散风寒药和发散风热药  
如果出现口渴，咽痛，舌质红等热性特征，可服用菊花，桑叶，薄荷等做成茶  
解表药一般不宜用于阴虚发热，久病体虚及失血的患者
而且煎煮时不宜久煎以免降低药效。
