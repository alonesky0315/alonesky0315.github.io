#### 利用浏览器截取网页保存为图片（谷歌内核浏览器都可以）
1、点击f12 

2、按住ctrl+shift+p 跳出输入框 输入cap 

> 截取整个网页(最高16384像素)有下一页的不行 ，下拉加载的 可以提前 加载完 然后截取

```
capture full size screenshot
```

>节点截取 配合 f12控制台的 选中 区域使用（ctrl+shift+p） 

```
capture node screenshot 
```

> 普通截图 当前屏幕 

```
capture screenshot
```
